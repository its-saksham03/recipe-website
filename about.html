About This Website

A little story behind every recipe.

Welcome to Delicious Recipes — a handcrafted collection of recipes that celebrate the beauty of home cooking. Whether you're a beginner or a seasoned home chef, there's something here for you.

This website was built as part of a Web Development assignment using pure HTML and CSS only — no JavaScript. It demonstrates how far thoughtful design and clean markup can take a project.

The site features 5 recipes spanning different cuisines and categories: Italian, Indian, and classic Western desserts. Each recipe page includes full ingredient lists, step-by-step instructions, prep time, and difficulty level.

Made with passion by Saksham — Web Development Student.
